# Screen-Spezifikation

## 1. Onboarding
Ziel: Sprache, Schriftgröße, Lernniveau und erste Präferenzen erfassen.

Pflichtinhalte:
- Begrüßung
- Auswahl der Ausgangssprache
- Auswahl des Lernniveaus
- Accessibility-Einstellungen
- Optional: Tagesziel

## 2. Learn Home
Ziel: Einstiegspunkt für alle Lernmodi.

Muss enthalten:
- Kategorien / Lernsets
- Einstieg in Lernkarten
- Einstieg in Prüfungsmodus
- Einstieg in Grammatik
- Einstieg in Satzbau
- Einstieg in Tagesplan
- Quick Stats

## 3. Learning Card
- russisches Wort / Hilfssprache
- deutsches Wort
- Artikel / Wortart / Beispiel
- Audio / TTS
- markieren als gelernt / schwierig
- nächste / vorherige Karte

## 4. Review Quiz
- Frage pro Karte
- mehrere Optionen oder Eingabe
- Bewertung korrekt / falsch
- Wiederholungslogik
- Session-Ergebnis

## 5. Speak Home
- Übersicht über Aussprachetraining
- Einstieg in Worttraining
- Einstieg in KI-Unterhaltung
- letzter Fortschritt

## 6. Aussprachetraining
- Zielwort
- Laut-Hinweise
- Shadowing-Anleitung
- Aufnahme-Button
- Wiedergabe
- Bewertung mit Score, Feedback und Tipps
- markieren als geschafft / erneut üben

## 7. KI-Unterhaltung
- Szenarioauswahl: Cafe, Arzt, Vorstellung, Einkauf, Arbeit, Behörde
- kurze Zielbeschreibung
- Tutor-Nachricht
- Nutzerantwort per Text, optional Sprache
- Feedback: Grammatik, Natürlichkeit, bessere Antwort, Vokabelvorschläge
- Gespräch fortsetzen

## 8. Grammatik
- Lektionen nach Niveau
- Erklärung
- Beispiele
- Mini-Quiz
- Bewertung und Punkte

## 9. Satzbau
- Wortbausteine oder Satzteile ordnen
- Feedback korrekt / falsch
- Erklärung der Lösung
- Punkte / Streak / Bonus

## 10. Prüfungsmodus
- Level A1, A2, B1, optional B2
- Zeitlimit
- gemischte Aufgabentypen
- Ergebnis pro Teilbereich
- Empfehlung, was als Nächstes gelernt werden soll

## 11. Dashboard
- Streak
- Punkte
- gelernte Wörter
- Review-Genauigkeit
- letzte Aktivität
- empfohlene nächste Schritte

## 12. Einstellungen
- Sprache
- Schriftgröße
- Dark Mode
- Notifications
- Audio / TTS
- Datenschutz / Konto / Sync
