# Design System

## Leitlinien
- sehr lesbar
- große Touch-Flächen
- ruhiges Layout
- mobile-first
- klare visuelle Hierarchie

## Farben
- hoher Kontrast
- definierte Tokens für background, foreground, muted, primary, success, warning, error

## Typografie
- große Überschriften
- großzügige Zeilenhöhe
- konsistente Größen je Hierarchiestufe

## Komponenten
- PrimaryButton
- SecondaryButton
- StatCard
- LearningCard
- QuizOption
- AudioButton
- RecordButton
- FeedbackCard
- ScenarioCard
- AchievementBadge

## Accessibility
- alle Buttons mit accessibility labels
- Screenreader-kompatible Texte
- reduzierte Animationen unterstützen
- Farbkontrast AA oder besser
