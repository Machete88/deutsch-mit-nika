# Umsetzungsplan für Manus

## Phase 1 - Stabilisierung
1. Projekt installieren und Build zum Laufen bringen
2. Imports, Routen und TypeScript-Fehler bereinigen
3. Storage und Auth säubern
4. UI-Sprache vereinheitlichen

## Phase 2 - Kernfunktionen fertigstellen
1. Learn / Review stabilisieren
2. Grammatik-Quiz produktionsreif machen
3. Satzbau-Punkte und Persistenz finalisieren
4. Dashboard auf echte Daten umstellen

## Phase 3 - Sprechen und KI
1. Aufnahmefluss stabil machen
2. STT integrieren
3. Ausspracheanalyse kapseln
4. KI-Unterhaltung mit echten Tutor-Antworten verbinden

## Phase 4 - Backend und Sync
1. echte Router und Services
2. Nutzerfortschritt synchronisieren
3. Fehler-Handling und Offline-Fallbacks

## Phase 5 - Release-Vorbereitung
1. Android testen
2. Smoke-Tests
3. Performance und Asset-Bereinigung
4. Release Build
